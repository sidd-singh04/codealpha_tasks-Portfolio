# Portfolio Website

## Description
A personal portfolio website built using HTML, CSS, and JavaScript. It showcases my skills, services, projects, and provides an interactive contact section for potential clients or recruiters. Designed to reflect a clean, modern, and responsive interface.

## Features
- Home section with an introduction and social links.
- About section highlighting background, skills, and experience.
- Skills section displaying technical expertise in frontend and backend technologies.
- Services section outlining UI/UX, frontend, and backend capabilities.
- Projects section with cards for multiple web projects and live links.
- Contact form for inquiries and direct communication.
- Fully responsive design for mobile, tablet, and desktop screens.
- Interactive navigation menu with smooth scrolling and animations.

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Tools: VS Code, Git & GitHub, FontAwesome
